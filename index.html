Table Wait

Mongo, Express, Angular, Node, Socket

Restaurant Waiting App gives restaurants a different way to handle their wait list. Instead of passing out expensive pagers or using a mic to shout out names, this app allows the restaurant to text or call customers to notify them that their table is ready. This app uses a twilio api that handles all calls or texts to customers. Users are allowed to add customers, text them when their table is ready, and check them in when they arrive. Wait times are updated in realtime using sockets.
